# Hero Image

Please add your hero image here as `hero-placeholder.jpg`.

**Recommended specifications:**
- Dimensions: 1600x1000px (16:10 aspect ratio)
- Format: JPEG or WebP
- Quality: High quality for best results
- Subject: Abstract, ethereal, light-focused imagery
- Color palette: Soft whites, light blues, pastels to match the ethereal theme

The frosted glass effect works best with images that have interesting visual details and varied tones.
